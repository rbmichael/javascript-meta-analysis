JavaScript Meta-Analysis (jsMA)

This software gratefully makes use of the following technologies: jStat, jQuery, goalSeek.js

In addition, much of the inspiration for the software comes from Exploratory Software for Confidence Intervals (ESCI)

Meta-analysis of single means (required: M, SD, N)
Meta-analysis of difference between two independent means (required: M1, SD1, N1, M2, SD2, N2)
Meta-analysis of Cohen's d between two independent means (required: d, N1, N2)
Meta-analysis of Pearson's r correlations (required: r, N)
